Css Inline

Bom pra explicar existem 3 jeitos de usar o css na sua página HTML. O primeiro é oque vou abordar
nesse aquivo. O Css Inline funciona Assim, eu tenho uma tag <p>, dentro dessa tag
depois do p, ou do h1 vamos colocar a palavra style, essa palavra vem acompanhada com ="", dentro
das aspas você vai colocar a estilização como color:red; que deixa o texto vermelho. A desvantagem de usar a Css Inline é que pra modificar 5 páragrafos com a mesma configuração você precisa fazer um por um. Ai entra o proximo método.
