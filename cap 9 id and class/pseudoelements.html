vamos falar de pseudo elementos.

bom basicamente é uma palavra chave que modifica partes especificas do elemento
selecionado. w3schools : Um pseudo-elemento CSS é uma palavra-chave adicionada a um seletor
que permite que você estilize uma parte específica do elemento selecionado.
Por exemplo, o pseudo-elemento ::first-line aplica o estilo apenas na primeira linha de um parágrafo.
