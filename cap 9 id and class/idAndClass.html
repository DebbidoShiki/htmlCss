oque você faria? Bom provavelmente usaria uma configuração inline de css

pra isso serve o id e o class, mas calma que não é tudo igual.
O id é uma identificação unica pra uma tag só e o class pode ser usado pra varias tags ao mesmo tempo.
Por exemplo eu que que o primeiro h2 tenha um sublinhado eu coloco o id e configuro usando a #.
Pra fazer a configuração é preciso colocar #(nome do id no html, que fica dentro da tag de abertura.)
ou . (configura a class e coloca o nome da class aqui também e ela fica dentro da tag de abertura.)
ou seja eu monto assim <h2 id="hh"> style="#hh{text-decoration:underline;}" e se for pra 2 h2 é só fazer assim: <h2 class="hh">
<h2 class.="hh"> style=".hh{text-decoration:underline;}.
E da pra por outra class ou id na mesma tag.
