Vamos falar sobre Pseudos Classe

de forma rápida as pseudo classes são formas
de definir um estado de uma tag e estilizar ela é bem mais ou menos isso mas a w3schools pode falar melhor
que eu: Uma pseudoclasse CSS é uma palavra-chave adicionada a
um seletor que permite estilizar um estado específico do(s) elemento(s) selecionado(s).
Por exemplo, a pseudoclasse pode ser usada para selecionar um botão quando o ponteiro
do usuário passa sobre o botão e esse botão selecionado pode então ser estilizado

Pra usar colocamos qual tag queremos alterar e coloca : e {} assim. p:hover{color:red;}
e pesquise oque cada pseudo classe faz.
